# Akka Persistence Cassandra

The Akka Persistence Cassandra plugin allows for using [Apache Cassandra](https://cassandra.apache.org) as a backend for [Akka Persistence](https://doc.akka.io/libraries/akka-core/2.10/persistence.html) and [Akka Persistence Query](https://doc.akka.io/libraries/akka-core/2.10/persistence-query.html). It uses [Alpakka Cassandra](https://doc.akka.io/docs/alpakka/10.0/cassandra.html) for Cassandra access which is based on the [Datastax Java Driver](https://docs.datastax.com/en/developer/java-driver/4.14/).
