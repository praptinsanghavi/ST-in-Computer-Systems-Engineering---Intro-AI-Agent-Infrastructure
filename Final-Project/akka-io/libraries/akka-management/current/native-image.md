# Building Native Images • Akka Management

> **Summary:** Akka Management is a suite of tools for operating Akka Clusters.

## Content

New to Akka? Start with the [Akka SDK](https://doc.akka.io/).

# Building Native Images

Building native images with Akka Management is supported out of the box for the following modules:

- akka\-management
- akka\-management\-cluster\-bootstrap
- akka\-management\-cluster\-http
- akka\-discovery\-kubernetes\-api
- akka\-lease\-kubernetes
- akka\-rolling\-update\-kubernetes

Other modules can likely be used but will require figuring out and adding additional native\-image metadata.

For details about building native images with Akka in general, see the [Akka Documentation](https://doc.akka.io/libraries/akka-core/current/additional/native-image.html).
