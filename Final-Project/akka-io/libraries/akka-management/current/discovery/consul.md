# Consul • Akka Management

> **Summary:** Akka Management is a suite of tools for operating Akka Clusters.

## Content

New to Akka? Start with the [Akka SDK](https://doc.akka.io/).

## Consul

The akka\-discovery\-consul module had upstream security issues and was not maintained, it has therefore been dropped from Akka Management. The last published version of this module was with Akka Management 1\.5\.0\.
