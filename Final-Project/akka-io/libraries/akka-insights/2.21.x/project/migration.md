# Migration guide

This is a guide for migrating from Akka Insights (Cinnamon) **2\.20** to Akka Insights (Cinnamon) **2\.21**.

## Configuration for OpenTelemetry Events / Logs

The config under `cinnamon.opentelemetry.events` have moved to `cinnamon.opentelemetry.logs`.

## Akka License key

This [section of our Akka documentation](https://doc.akka.io/docs/akka/current/general/configuration.html#license-key) describes how to configure the license key, which is required to run in production.
