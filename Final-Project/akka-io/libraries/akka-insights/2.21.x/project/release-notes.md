## 📦 Akka Insights 2\.21\.x

Migration Guide: [Migration guide for Akka Insights 2\.21](migration.html)

This release of Akka Insights (Cinnamon) adds exclusive support for Akka 24\.10\.

Note
From Cinnamon 2\.18 onward, each Cinnamon release will only support a single Akka release.

The following Akka 24\.10 modules are supported for this Cinnamon release:

- Akka (core) \>\= 2\.10\.0
- Akka HTTP \>\= 10\.7\.0
- Akka gRPC \>\= 2\.5\.0
- Akka Projections \>\= 1\.6\.0
- Alpakka Kafka \>\= 7\.0\.0

On the following versions of Scala and Java:

- Scala 2\.13 and 3\.3
- Java 11, 17, and 21

#### Akka license key

This is the first release of Cinnamon that [requires license key for use of Akka in production](https://akka.io/blog/news/2024/10/02/akka-2.9.6-released).
