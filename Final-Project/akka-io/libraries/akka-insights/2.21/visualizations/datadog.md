# Datadog

We have partnered with [Datadog](https://www.datadoghq.com) to provide a plugin for Akka Insights for Akka with Datadog.

Note
Make sure you [configure Akka Insights for Datadog](../plugins/datadog/datadog.html) before proceeding.

## Datadog dashboard for Akka

The Datadog integration gives you access to a predefined dashboard created by us to help you jump\-start the monitoring of your Akka applications.

To use the predefined dashboard, login to your Datadog account, go to *Dashboards* and then *Add Integration* under *Integrated Dashboards*. Select the *Lightbend Reactive Platform* integration in the displayed list. Once the integration is enabled, the *Lightbend – Akka Overview* dashboard will be available.
