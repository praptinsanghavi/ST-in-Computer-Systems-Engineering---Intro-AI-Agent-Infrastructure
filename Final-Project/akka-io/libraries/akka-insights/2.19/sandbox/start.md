# Developer sandbox

Akka Insights provides a complete developer sandbox environment that contains everything you need to explore your application. The purpose of the developer sandbox is to make it easy to get started and explore the capabilities of Akka Insights.

Warning
The developer sandbox is only meant for development and testing purposes and **SHOULD NOT** be used in production.

Cinnamon provides three developer sandboxes:

- [Prometheus](prometheus-sandbox.html)
- [Elasticsearch](elastic-sandbox.html)
- [Graphite](graphite-sandbox.html)

The developer sandboxes utilize [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/), which need to be installed before running the developer sandbox containers.

The following sections contain information about how to configure, how to run, and explore the sandboxes.
