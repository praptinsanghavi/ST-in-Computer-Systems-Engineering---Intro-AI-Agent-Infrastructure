# Getting Started Guide

By now you should understand the fundamental concepts of how a Projection works by reading about its [use cases](../use-cases.html). This guide will briefly describe the basic components of a Projection and instruct you step\-by\-step on how to build a functioning application.

The example used in this guide is based on a more complete application that is part of the [Microservices with Akka tutorial](https://doc.akka.io/libraries/guide/microservices-tutorial/). It builds a full [CQRS](../use-cases.html#command-query-responsibility-segregation-cqrs-) (Command Query Responsibility Segregation) ES (Event Sourcing) system using a combination of features from the Akka toolkit.
