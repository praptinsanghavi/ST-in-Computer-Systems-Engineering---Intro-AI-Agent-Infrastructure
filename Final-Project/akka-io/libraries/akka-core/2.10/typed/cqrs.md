# CQRS • Akka core

> **Summary:** Akka is a toolkit for building highly concurrent, distributed, and resilient message-driven applications for Java and Scala.

## Content

New to Akka? Start with the [Akka SDK](https://doc.akka.io/).

# CQRS

[EventSourcedBehavior](persistence.html)s along with [Akka Projections](https://doc.akka.io/libraries/akka-projection/current/) can be used to implement Command Query Responsibility Segregation (CQRS). The [Microservices with Akka tutorial](https://doc.akka.io/libraries/guide/microservices-tutorial/) explains how to use Event Sourcing and Projections together. For implementing CQRS using [DurableStateBehavior](durable-state/persistence.html), please take a look at the corresponding [CQRS](durable-state/cqrs.html) documentation.
