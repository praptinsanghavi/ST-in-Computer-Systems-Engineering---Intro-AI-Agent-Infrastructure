# CQRS • Akka core

> **Summary:** Akka is a toolkit for building highly concurrent, distributed, and resilient message-driven applications for Java and Scala.

## Content

New to Akka? Start with the [Akka SDK](https://doc.akka.io/).

# CQRS

[DurableStateBehavior](persistence.html)s along with [Akka Projections](https://doc.akka.io/libraries/akka-projection/current/) can be used to implement Command Query Responsibility Segregation (CQRS). For implementing CQRS using [EventSourcedBehavior](../persistence.html), please take a look at the corresponding [CQRS](../cqrs.html) documentation.
