- [StringHasher](struct.StringHasher.html "struct akka_persistence_rs::jdk::StringHasher")A hasher for hashing strings conformant with
[https://docs.oracle.com/javase/8/docs/api/java/lang/String.html\#hashCode.](https://docs.oracle.com/javase/8/docs/api/java/lang/String.html#hashCode.).

Rust [String](https://doc.rust-lang.org/1.75.0/alloc/string/struct.String.html "struct alloc::string::String") is hashed.
