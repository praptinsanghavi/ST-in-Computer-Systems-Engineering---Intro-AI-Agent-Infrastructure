```
public final class JavaFlowSupport
extends java.lang.Object
```

For use only with \`JDK 9\+\`.

 Provides support for \`java.util.concurrent.Flow.\*\` interfaces which mirror the Reactive
 Streams interfaces from \`org.reactivestreams\`. See [reactive\-streams.org](http//www.reactive-streams.org/).
