# Package akka.annotation

- Annotation Types Summary 
| Annotation Type | Description |
| [ApiMayChange](ApiMayChange.html "annotation in akka.annotation") | Marks APIs that are meant to evolve towards becoming stable APIs, but are not stable APIs yet. |
| --- | --- |
| [DoNotInherit](DoNotInherit.html "annotation in akka.annotation") | Marks APIs that are designed under a closed\-world assumption for and are NOT meant to be extended  by user\-code. |
| [InternalApi](InternalApi.html "annotation in akka.annotation") | Marks APIs that are considered internal to Akka and may change at any point in time without any  warning. |
| [InternalStableApi](InternalStableApi.html "annotation in akka.annotation") | Marks APIs that are considered internal to Akka and should not be accessed by user code but that  are used across Akka project boundaries and therefore shouldn't be changed without considering  possible usage outside of the Akka core modules. |
