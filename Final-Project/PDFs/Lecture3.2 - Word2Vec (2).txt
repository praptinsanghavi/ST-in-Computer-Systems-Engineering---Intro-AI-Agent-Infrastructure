Word2vec is similar to an autoencoder, encoding each word in a vector, but

Ôā®

either using context to predict a target word (a method known as continuous

or using a word to predict a target context, which is called skip-gram. We

datasets.

The components of the vector are adjusted
8

Ôā®

When the feature vector assigned to a word cannot be used to accurately
predict that word‚Äôs context, the components of the vector are adjusted.

Ôā®

Each word‚Äôs context in the corpus is the teacher sending error signals back
to adjust the feature vector.

Ôā®

The vectors of words judged similar by their context are nudged closer
together by adjusting the numbers in the vector.

Model overview
9
Ôā®

We have to feed some strange neural network with some pairs of words but we
can‚Äôt just do that using as inputs the actual characters, we have to find some
way to represent these words mathematically so that the network can process
them.

Ôā®

One way to do this is to create a vocabulary of all the words in our text and
then to encode our word as a vector of the same dimensions of our vocabulary.

Ôā®

Each dimension can be thought as a word in our vocabulary.

Ôā®

So, we will have a vector with all zeros and a 1 which represents the
corresponding word in the vocabulary.

Ôā®

This encoding technique is called one-hot encoding.

In word2vec, a distributed representation of a word is used
13

Ôā®

Ôā®

Each word is represented by a distribution of weights across those

Ôā®

So instead of a one-to-one mapping between an element in the vector and a
word, the representation of a word is spread across all the elements in the
vector, and each element in the vector contributes to the definition of many
words.

Word2vec uses a single hidden layer
30

Word2vec uses a single hidden layer, fully connected neural network,
Ôā® The neurons in the hidden layer are all linear neurons.
Ôā® The input layer is set to have as many neurons as there are words in the
vocabulary for training.
Ôā® The hidden layer size is set to the dimensionality of the resulting word vectors.
Ôā® The size of the output layer is same as the input layer.
Ôā® Thus, if the vocabulary for learning word vectors consists of V words and N to be
the dimension of word vectors, the input to hidden layer connections can be
represented by matrix WI of size VxN with each row representing a vocabulary
word.
Ôā® In same way, the connections from hidden layer to output layer can be described
by matrix WO of size NxV.
Ôā® In this case, each column of WO matrix represents a word from the given
vocabulary.
Ôā®

Modification
43
Ôā®

We need few additional modifications to the basic skip-gram model which are
important for making it feasible to train.

Ôā®

Running gradient descent on a neural network that large is going to be slow.

Ôā®

And to make matters worse, you need a huge amount of training data to tune
that many weights and avoid over-fitting.

Ôā®

Millions of weights times billions of training samples means that training this
model is going to be a beast.

Ôā®

For that, authors have proposed two techniques called subsampling and
negative sampling in which insignificant words are removed and only a specific
sample of weights are updated.
